## 🧰 Tech Stack

### 🟦 Minecraft Plugins
- **Java** - Bukkit/Spigot/Paper plugin development  
- **YAML** - Configurations  
- **Gradle / Maven** - Build systems  

### 🟨 Little-Community (Current Project)
- **JavaScript**  - Core logic  
- **React Native** - Mobile client  
- **React + Tailwind** - Web client  
- **Express.js** - Backend API  
- **MongoDB** - Database  
- **WebSocket** - Real‑time multiplayer
- **Hosting** - (TBD)

### 🟧 2D Graphic Utility (Game Library)
- **JavaScript** - Rendering engine  
- **HTML Canvas** - Graphics layer
